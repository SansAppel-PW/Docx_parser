## 🚀 使用指南

### 批量处理
```bash
python docx_parser.py Files/PLM2.0
# 结果保存在: parsed_docs/PLM2.0_parsed/
```

### 单文件处理
```bash
python docx_parser.py "Files/PLM2.0/BOM 审核申请.docx"
# 结果保存在: parsed_docs/single_files/BOM 审核申请/
```
